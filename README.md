# PATHGENIE AI
Navigate Your Future with AI-Powered Career Insights and Skill Intelligence.
# Abstract
PathGenie AI is an intelligent career recommendation system developed to help students and beginners identify suitable technology domains based on their skills, interests, and technical knowledge. The application uses Machine Learning techniques to analyse user responses and provide real-time career recommendations in areas such as Web Development, Machine Learning, Python Development, and Cyber Security.
The project is designed using Python and Streamlit with a clean and beginner-friendly interface. It demonstrates the practical implementation of Artificial Intelligence, data preprocessing, and prediction systems while helping users make informed career decisions through an interactive and easy-to-use platform.
# Introduction
In the modern technology industry, students and beginners are often confused about selecting the right career path because of the large number of available domains such as Artificial Intelligence, Data Science, Web Development, and Cyber Security. Many learners do not have proper guidance to understand which field matches their skills and interests. Choosing the wrong domain can affect learning progress, confidence, and future career growth.
To solve this problem, PathGenie AI was developed as a smart career recommendation system that uses Machine Learning to analyse user skills, interests, and technical knowledge. The application collects user responses through interactive questionnaires and predicts the most suitable career domain based on the analysed data.
The project is developed using Python, Streamlit, and Machine Learning libraries to create a simple, interactive, and beginner-friendly platform. Along with providing career recommendations, the project also demonstrates practical implementation of important Data Science concepts such as data preprocessing, feature engineering, model training, and intelligent prediction systems.

## 🛠️ Technologies Used
The following technologies and tools were used in the development of the **PathGenie AI** project. Each technology plays an important role in building the system, processing data, training the Machine Learning model, and generating intelligent career recommendations for users.
| Technology | Purpose in the Project |
|---|---|
| **Python** | Python is used as the core programming language of the project. It provides powerful libraries and frameworks for Machine Learning, data processing, and web application development. The complete backend logic and prediction system are developed using Python. |
| **Streamlit** | Streamlit is used to build the interactive web application interface. It allows users to answer questions, submit inputs, and receive career recommendations instantly through a simple and user-friendly dashboard. |
| **Pandas** | Pandas is used for data handling and preprocessing. It helps in reading datasets, cleaning and organizing data, and preparing user input for Machine Learning prediction. |
| **NumPy** | NumPy is used for numerical computations and efficient array operations. It supports data preprocessing and improves performance during model calculations. |
| **Scikit-learn** | Scikit-learn is used to build, train, and evaluate the Machine Learning model. It helps the system predict suitable career domains based on user interests, skills, and responses. |
| **Joblib** | Joblib is used for saving and loading trained Machine Learning models efficiently. It enables faster predictions without retraining the model every time the application runs. |
| **Pickle / PKL Models** | PKL files are used to store trained models, encoders, and processed data. These saved files allow the application to directly load the trained components during prediction. |
| **Machine Learning** | Machine Learning is the core technology behind the project. It enables the system to analyze user skills, interests, and preferences to generate intelligent and personalized career recommendations. |

# How the Project Works
The PathGenie AI system follows a simple and intelligent workflow to analyse user inputs and generate personalised career recommendations.
## Step 1 – User Accesses the Application
The user opens the Streamlit-based web application through the browser. The application provides a clean, interactive, and user-friendly interface for career assessment.
## Step 2 – User Provides Information
The system collects user responses through a set of carefully designed questions related to:
Skills and strengths
Personal interests
Technical knowledge
Career preferences and goals
These inputs help the system understand the user’s profile more accurately.
## Step 3 – Data Processing and Analysis
The collected responses are preprocessed and converted into a format suitable for Machine Learning prediction. The trained AI model then analyses the user data to identify patterns and matching career domains.
## Step 4 – Career Prediction
Based on the analysis, the Machine Learning model predicts the most suitable career path for the user. The system generates intelligent and personalised career recommendations according to the user’s skills and interests.
## Step 5 – Result Display
The recommended career domain is displayed instantly on the web application dashboard, allowing the user to explore suitable career opportunities and make informed decisions.
## 🚀 Installation Guide
Follow the steps below to set up and run the **PathGenie AI** project on your system.
### 1️⃣ Clone the Repository
git clone https://github.com/M-Harshitha-2005/PathGenie-_AI.git
### 2️⃣ Navigate to the Project Directory
cd PathGenie-_AI
### 3️⃣ Install Required Dependencies
pip install -r requirements.txt
### 4️⃣ Run the Streamlit Application
streamlit run app.py
### 5️⃣ Open the Application in Browser
After running the above command, Streamlit will automatically generate a local URL and open the application in your default web browser.
Example:
https://pathgenie-ai.onrender.com/
You can now interact with the **PathGenie AI** career recommendation system.

# Machine Learning Concept 
The project uses a trained Machine Learning model to analyse user responses and predict the most suitable career domain. The complete workflow follows standard Data Science and Machine Learning practices.
ML Workflow
## Data Collection
The dataset contains skill-based and interest-based information related to different technology domains.
## Data Preprocessing
The collected data is cleaned and transformed before training the model.
This includes:
Handling missing values
Feature selection
Encoding categorical values
Data formatting and transformation
## Feature Engineering
Important features are prepared and structured to improve prediction performance.
## Model Training
The Machine Learning model is trained using processed data to learn patterns between user skills and career domains.
## Prediction System
When users answer questions in the application, the trained model analyses the responses and predicts the most suitable career option.
## Recommendation Generation
